## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### preview the release environment effect
```
npm run preview
```

### preview the release environment effect + static resource analysis
```
npm run report
```

### Lints and fixes files
```
npm run lint
```
